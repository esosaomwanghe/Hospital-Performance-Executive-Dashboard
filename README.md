# Hospital Performance Executive Dashboard

Fully interactive Tableau dashboard monitoring:
- Patient readmission rates & cost per discharge
- Department efficiency KPIs
- 12-month forecasting
- Top N performers + dynamic filters

**Advanced features**  
- Level of Detail (LOD) expressions  
- Parameters & calculated fields  
- Forecast confidence intervals
